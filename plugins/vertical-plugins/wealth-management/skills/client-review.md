# Skill: client-review
**Command:** `/client-review`

## Purpose
Generate a quarterly or annual client portfolio review report.

## Sections
1. **Portfolio Performance** — absolute and benchmark-relative returns (1M, 3M, YTD, 1Y, 3Y, 5Y)
2. **Asset Allocation** — current vs. target allocation, drift analysis
3. **Holdings Summary** — top holdings, sector/geographic exposure
4. **Risk Metrics** — volatility, Sharpe ratio, max drawdown, beta
5. **Goal Progress** — projected vs. target wealth at retirement/goal date
6. **Recommendations** — rebalancing, tax opportunities, goal adjustments
7. **Market Commentary** — brief macro context

## Output Format
- PDF-ready structured report
- Charts: allocation pie, performance vs. benchmark, goal projection
- Action items list

## Disclaimer
For informational purposes only. Not investment advice. Consult your advisor.
