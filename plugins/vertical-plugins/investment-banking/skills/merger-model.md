# Skill: merger-model
**Command:** `/merger-model`

## Purpose
Build an accretion/dilution analysis for a proposed merger or acquisition.

## Steps
1. Collect acquirer and target standalone financials (EPS, shares, net income).
2. Set transaction assumptions: offer price, premium, form of consideration (cash/stock/mix).
3. Calculate purchase price and goodwill.
4. Model financing: new debt issued, shares issued, cash used.
5. Apply synergies (revenue and cost) with phasing assumptions.
6. Compute pro forma combined net income and EPS.
7. Calculate accretion/(dilution) vs. acquirer standalone EPS.
8. Run sensitivity on synergies, premium, and financing mix.

## Output Format
- Transaction summary (sources & uses, ownership split)
- Pro forma income statement
- EPS accretion/dilution table
- Synergy sensitivity matrix
