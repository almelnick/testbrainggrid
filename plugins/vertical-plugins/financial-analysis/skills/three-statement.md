# Skill: three-statement
**Command:** `/3-statement`

## Purpose
Build an integrated 3-statement financial model (Income Statement, Balance Sheet, Cash Flow Statement).

## Steps
1. Input historical financials (3–5 years of IS, BS, CF).
2. Build revenue model with growth assumptions by segment.
3. Project operating expenses as % of revenue or absolute.
4. Derive EBITDA, EBIT, EBT, Net Income.
5. Build working capital schedule (AR, inventory, AP days).
6. Build PP&E and capex schedule.
7. Construct cash flow statement (indirect method).
8. Balance sheet must balance: Assets = Liabilities + Equity.
9. Build debt and interest schedule (revolver as plug).
10. Validate model integrity: BS balance, CF ties to IS/BS.

## Output Format
- 3-statement model with 3-year history + 5-year projections
- Key ratio dashboard (margins, leverage, coverage, returns)
- Assumption summary
