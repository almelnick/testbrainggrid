# Skill: lbo-model
**Command:** `/lbo`

## Purpose
Build a leveraged buyout (LBO) model to assess private equity return potential.

## Steps
1. Set entry assumptions: entry multiple, purchase price, transaction fees.
2. Build debt structure: senior secured, subordinated, total leverage turns.
3. Project EBITDA over 5-year hold period.
4. Model debt repayment schedule (cash sweep, mandatory amortization).
5. Calculate exit at assumed multiple and hold period.
6. Compute sponsor returns: IRR and MOIC across exit multiple / EBITDA growth scenarios.
7. Run sensitivity analysis on entry multiple, exit multiple, and leverage.

## Output Format
- Sources & uses table
- Debt schedule
- Returns summary (IRR, MOIC)
- Sensitivity matrix

## Disclaimer
For informational purposes only. Not investment advice.
