# Skill: dcf-model
**Command:** `/dcf`

## Purpose
Build a discounted cash flow (DCF) valuation model for a company.

## Steps
1. Collect or project 5–10 years of free cash flow (FCF) forecasts.
2. Calculate WACC: cost of equity (CAPM), cost of debt (after-tax), capital structure weighting.
3. Discount projected FCFs to present value using WACC.
4. Calculate terminal value using Gordon Growth Model or exit multiple method.
5. Sum PV of FCFs + PV of terminal value = Enterprise Value.
6. Bridge to equity value: EV – net debt + cash.
7. Divide by diluted share count for implied share price.
8. Run sensitivity tables on WACC and terminal growth rate / exit multiple.

## Output Format
- WACC build
- FCF projection table
- DCF bridge (EV → equity value → per share)
- Sensitivity tables (2 scenarios minimum)
