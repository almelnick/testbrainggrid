# Skill: thesis-tracker
**Command:** `/thesis`

## Purpose
Track an investment thesis over time — compare original assumptions to actual results and flag thesis breaks.

## Steps
1. Load original initiation thesis: key assumptions, price target, timeline.
2. Pull actual results for each tracked metric (revenue growth, margins, FCF, market share).
3. Score each assumption: On Track / Ahead / Behind / Thesis Break.
4. Calculate implied price target revision based on estimate changes.
5. Assess overall thesis integrity: Intact / Weakening / Broken.
6. Draft thesis update memo.

## Output Format
- Thesis scorecard: assumption, original estimate, current actual, status
- Estimate revision table (vs. initiation)
- Updated price target with delta explanation
- Thesis update memo (1 page)

## Disclaimer
Draft only. Not a recommendation. Requires analyst review.
