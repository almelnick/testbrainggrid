# Skill: ic-memo
**Command:** `/ic-memo`

## Purpose
Draft an Investment Committee (IC) memo for a private equity deal.

## Sections
1. **Executive Summary** — deal overview, recommendation, key metrics (entry multiple, leverage, IRR)
2. **Business Description** — products, customers, business model, unit economics
3. **Market Opportunity** — TAM, growth, competitive dynamics
4. **Investment Thesis** — 3–5 key value creation levers
5. **Financial Analysis** — historical performance, LBO model summary, returns sensitivity
6. **Risks & Mitigants** — top 5 risks with mitigation strategies
7. **Management Assessment** — team track record, retention plan
8. **Transaction Structure** — price, financing, key terms, timeline
9. **Value Creation Plan** — 100-day plan, EBITDA bridge to exit

## Output Format
- Structured memo (10–15 pages equivalent)
- Returns summary table (IRR, MOIC across scenarios)
- Risk matrix

## Disclaimer
Draft only. Requires senior partner and IC review before vote.
