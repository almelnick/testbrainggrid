# Skill: portfolio-monitor
**Command:** `/portfolio`

## Purpose
Generate a portfolio company performance summary for quarterly LP reporting.

## Steps
1. Ingest quarterly financials for each portfolio company.
2. Compare actuals to budget and prior year: Revenue, EBITDA, EBITDA margin, cash.
3. Assess KPIs specific to the company/sector (ARR, NRR, bookings, units, etc.).
4. Update fair value mark based on current trading comps and internal model.
5. Track value creation plan milestones (e.g., new hires, product launches, M&A).
6. Produce quarterly update narrative suitable for LP report.

## Output Format
- Portfolio dashboard: company, revenue (actual vs. budget), EBITDA, FMV, MoIC
- Per-company narrative (150–200 words): highlights, concerns, outlook
- Value creation plan tracker

## Disclaimer
For LP reporting purposes. Fair value marks are estimates pending final quarterly close.
